# Code of Conduct

## Our Standards

This project is committed to providing a welcoming and inclusive environment for all contributors.

We expect all participants to:

- Be respectful and considerate
- Accept constructive feedback gracefully
- Focus on what is best for the community
- Show empathy towards others

## Scope

This applies to all project spaces including GitHub issues, pull requests, discussions, and any other forums.

## Enforcement

Project maintainers are responsible for clarifying standards and taking appropriate action in response to unacceptable behavior.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
